# 功能
![](https://github.com/user-attachments/assets/06cd01a6-f597-4411-bf32-d1a3a5f21a6b)
- 批量创建文件（文件夹）
- 音乐播放器（可有可无）
- 二维码生成器
- 随机点名器（很恶心的程序，不过是我的开山之作）
- PDF关键词分析
- 邮件功能（不强大，但勉强可用，毕竟我也不会写邮件客户端）
- 没有日历，但是可以显示当前时间（与当前时间误差1秒左右）
- 天气查询（后续加上）
